How To Use:
- Go on https://discord.com/developers
- Login 
- Create an Application (New Application Button)
- Add a Name and accept the Terms
- Copy the Application ID and paste it in the Config
- Click on Rich Presence
- Add the Image you want
- Now fill out the conf.config
- ("pic" : "NAME") is the Image-Name on the Rich Presence Tab
- Execute the programm

- YEY | Now you have your own Custom Rich Presence
